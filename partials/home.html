AngularJS REST Client

About this site

This is a coding project to explore various technologies to produce an enterprise content management system. I'm not sure if that's exactly accurate, but it sounds cool – I'm keep'n it. This project uses the following to create a light-weight web client in conjunction with an Enterprise Java REST endpoint managing user resources.

  • AngularJS
  • Bootstrap
  • JQuery
  • bower.io
  • node.js

    • These technologies were uses to create a dynamic client-side presentation, manage development packages (dependencies), and test locally.

  • J2EE JSR 370
  • Apache Maven

    • The Jersey project's reference impl. of JSR 370, the Java API for RESTful Web Services is used for creating URL endpoints for serving resources. Apache Maven is used for dependency management and build/deployment lifecycle.

  • GitHub
  • Google Appengine

    • GitHub pages for source code management and static file hosting, via gh-pages. Google's Appengine for API endpoint hosting and resource management.
